## Access

Hosted at: `https://wmestrinho.github.io/absolutelyplausible-business-plan/`

Open from any browser, any device, anywhere.

## Update

```bash
# Edit any .md file in docs/
# Commit and push to main
git add .
git commit -m "update: [what changed]"
git push

# GitHub Actions deploys automatically (~60 seconds)
```

## Local Preview

```bash
pip install mkdocs-material
mkdocs serve
# Open http://localhost:8000
```
